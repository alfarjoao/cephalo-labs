# Cephalo Labs — Website

## Setup

```bash
cd cephalo-labs
npm install
npm run dev
```

## Logos & Images

Place these files in `/public/logos/`:
- `cephalo-mark.png` → CEPHALO_LOGO.png (the octopus mark)
- `cephalo-writing.png` → CEPHALO_WRITING_LOGO.png
- `polypus-mark.png` → Polypus logo
- `pantheon-logo.png` → Pantheon Growth logo
- `titan-ai-mark.png` → TITAN AI logo
- `axiom-mark.png` → AXIOM logo
- `sovereign-mark.png` → SOVEREIGN logo

Place founder photo in `/public/images/`:
- `owner.png` → OWNER CEPHALO.png

## Pages

- `/` — Home
- `/about` — About Cephalo Labs
- `/products` — Products (Polypus)
- `/projects` — All projects
- `/projects/:slug` — Project detail
- `/founder` — João Alfar
- `/partners` — Pantheon Growth
- `/contact` — Contact

## Stack

React 18 + Vite + TypeScript + Tailwind + Framer Motion + Three.js
